Senior React Developer

Location: Remote (US / Canada)

Type: Full-Time

Job Description

We are seeking a Senior React Developer to join our frontend team. You will build high-quality, responsive web interfaces and design system components using React, Tailwind CSS, and TypeScript.

Responsibilities

  • Develop new user-facing features using React.js and Tailwind CSS.
  • Optimize application components for maximum performance across web-capable devices.
  • Collaborate with backend engineers to integrate Express.js REST APIs.
  • Write clean, documented, and reusable frontend components.

Requirements

  • 5+ years of experience as a Frontend Engineer.
  • Expert proficiency in React.js, JavaScript, and Tailwind CSS.
  • Strong familiarity with Git, GitHub, and Axios.
  • Experience with Node.js and Express.js is a plus.
